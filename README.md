## ğŸ“– Tentang Aplikasi

**Aplikasi Panjar (SIPANJOEL)** adalah sistem informasi berbasis web yang dikembangkan khusus untuk **Pengadilan Agama Pasuruan**. Aplikasi ini berfungsi untuk membantu masyarakat pencari keadilan dalam menghitung estimasi biaya panjar perkara secara mandiri, cepat, dan transparan berdasarkan radius lokasi tempat tinggal.

Aplikasi ini dilengkapi dengan fitur-fitur modern untuk memudahkan pengguna:

* ğŸ“ **Hitung Otomatis Berdasarkan Radius**
    <br>Kalkulasi biaya panjar yang akurat menyesuaikan dengan zona/radius lokasi pihak berperkara.
* ğŸ”Š **Text to Speech (Suara)**
    <br>Fitur aksesibilitas yang membacakan hasil perhitungan biaya (didukung oleh *ResponsiveVoice*).
* ğŸ“„ **Cetak PDF**
    <br>Hasil perhitungan dapat langsung dicetak atau disimpan dalam format dokumen digital (PDF) sebagai arsip.
